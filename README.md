Map OCR / AA
=============================

uses ocr to read map location text and display a icon on the map of its location
based on the older version of my map tracker that was hosted on bitbucket before the take over.

= improvements / changes.
1: Changed OCR engine (its no longer paid for)
2: simpler processing and app (we have addons for normal usage now)
3: updated to .net 10 from 7 as it had not been used in a long time
